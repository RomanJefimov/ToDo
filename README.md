# ToDo
ToDo это приложениеv(Планировщик задач), написаное преимущественно на TypeScript.
В нем можно создавать задачи, выбирая им приоритет, а также можно удобно сортировать их по степени выполнения (Что нужно сделать, В процессе выполнения, На стадии тестирования и Выполнено).
Для того чтобы приложение работало необходимо установить пакеты (команда: npm i), затем загрузить скрипты (команда: npm run build -- --watch).
После этого можно запускать приложение с помощью Live Server.
